# Early-Bound Types Generation

This project generates strongly-typed Dataverse entity classes (`EarlyBoundTypes.cs`) using **spkl**.

These classes allow you to work with Dataverse data in a safe and readable way (instead of using string-based attributes).

---

## What does it do?

* Converts Dataverse metadata into C# classes
* Enables strongly-typed access to entities
* Reduces runtime errors caused by typos in field names

---

## How it works

* The classes are generated using `earlybound.bat`
* Configuration is defined in `spkl.json`
* Output file: `EarlyBoundTypes.cs`
* The project targets multiple frameworks, so it can be used from both .NET Core and .NET Framework projects
* Other projects (e.g. tests) use these classes via **project reference**

---

## Important rules

* `EarlyBoundTypes.cs` is **auto-generated**
* It can be **completely overwritten**
* ❗ **Do not edit this file manually**
* Always regenerate using `earlybound.bat`

---

## When to regenerate

Run generation when:

* Dataverse schema changes (entities, fields, relationships)
* You need newly added attributes in code
* Something stops matching reality
